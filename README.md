## Project Overview

**CoinForge** is an idle/clicker game focused on massive capital accumulation. The player starts by manually clicking on a central coin to generate income, which must then be strategically reinvested in infrastructure and technological upgrades to automate production and maximize **Money Per Second (MPS)**.

### Key Features
* **Active Clicking:** Immediate capital generation through direct interaction.
* **Passive Income:** Construction of infrastructure that generates money automatically.
* **Exponential Scaling:** Cost and benefit systems that scale as the empire grows.
* **Economic Upgrades:** Upgrade tree to boost both manual clicks and structures.

### The Assets
The game is based on managing two main pillars to increase fortune:

| Basic Skill | Building | Special Skill | Physical Attribute |
| :--- | :--- | :--- | :--- |
| **Active** | Piggy Bank | **Basic Storage** | Initial manual boost |
| **Passive** | ATM Machine | **Auto-Withdrawal** | Light automation |
| **Passive** | Central Bank | **Asset Management** | Large-scale income |
| **Passive** | Digital Mine | **Crypto-Mining** | Advanced technology |
